
 Validador de Incoterms 2020

Herramienta interactiva para validar Incoterms según las reglas ICC 2020. Permite verificar compatibilidad con el medio de transporte, lugar de entrega y alertas críticas.

## ✅ Características
- Selección del **Incoterm** y medio de transporte.
- Validación de compatibilidad según reglas oficiales.
- Alertas especiales (ej. seguro obligatorio).
- Resumen con punto de riesgo y responsabilidades de exportación/importación.

> **Nota:** Esta herramienta es de carácter educativo y no sustituye asesoría legal ni interpretaciones oficiales de ICC.
